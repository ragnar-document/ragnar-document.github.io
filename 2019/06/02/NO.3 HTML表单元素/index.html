以下是使用Bootstrap进行表单验证的方法：

  • HTML表单验证通过CSS的两个伪类应用，:invalid以及:valid。它适用于<input>，<select>和<textarea>元素。
  • Bootstrap将:invalid和类的范围限定:valid为父.was-validated类，通常应用于<form>。否则，没有值的任何必填字段在页面加载时显示为无效。这样，您可以选择何时激活它们（通常在尝试提交表单之后）。
  • 作为备用，.is-invalid并且.is-valid类别可以用来代替伪类用于服务器端验证。他们不需要.was-validated父类。
  • 由于CSS的工作方式存在限制，我们无法（目前）<label>在没有自定义JavaScript帮助的情况下将样式应用于DOM中的表单控件之前。
  • 所有现代浏览器都支持约束验证API，这是一系列用于验证表单控件的JavaScript方法。
  • 反馈消息可以使用浏览器默认值（每个浏览器不同，通过CSS不可用）或我们的自定义反馈样式以及其他HTML和CSS。
  • 您可以使用setCustomValidityJavaScript 提供自定义有效性消息。
