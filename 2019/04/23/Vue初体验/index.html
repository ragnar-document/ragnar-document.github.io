    v-bind绑定元素特性props类似于一个自定义特性

    这里我们遇到了一点新东西。你看到的v-bind特性被称为指令。指令带有前缀v-，以表示它们是 Vue 提供的特殊特性。可能你已经猜到了，它们会在渲染的 DOM 上应用特殊的响应式行为。在这里，该指令的意思是：“将这个元素节点的title特性和 Vue 实例的message属性保持一致”。
